Welcome to strax's documentation!
=================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:



Strax is a processor for pulse-only digitization data, specialized for live data reduction at speeds of 50-100 MB(raw) / core / sec.
It includes numba-powered fast algorithms for low-level data processing, and an extension/plugin system to build more complete processors and analyses on top of this.
